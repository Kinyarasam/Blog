Definition of Terms

The following are some of the terms that may be worth learning as you learn more about how the web works. They may include;
  • Client
    • A client can be defined as a program that uses resources provided by the server.
  • Server
    • A Server can be defined as the program/special software that makes this resource available to the client and is responsible for memory locations.
  • DNS
    • The initials represent the Domain Name System. The DNS is responsible for translating human readable domain names into machine readable ip addresses.
  • Ip address
    • An ip address is a unique address that identifies a device on the internet or a local network.
  • ISP
    • ISP refers to the internet service provider. This may be a company that provides individuals, organizations or other companies access to the internet and other related services.
